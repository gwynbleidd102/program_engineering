Этот проект содержит три функции:

1. `is_palindrome` - проверяет, является ли строка палиндромом.
2. `is_correct_mobile_phone_number_ru` - проверяет корректность номера мобильного телефона в России.
3. `strip_punctuation_ru` - удаляет знаки препинания из строки на русском языке.

## Установка

Установите необходимые зависимости через терминал:

pip install -r requirements.txt
